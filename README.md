## Fastify + Lit + Rollup Boilerplate

### Application

#### Prepare project 

* clone this repo 
* copy `.env.exaple` and name to `.env`

#### Run application: 

Run `npm run start` or `yarn start`

### Frontend

Frontend project is placed in `app/website` directory

#### Dev:

Run `npm run start:watch` `yarn start:watch`
